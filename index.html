客户端核心功能

安全、高效的桌面密码管理

🔐

端到端加密

采用 AES-256-GCM 加密算法，所有数据本地存储，无需联网，杜绝数据泄露风险。

🔑

密码管理

安全存储网站、应用的账号密码，支持分组管理、快速搜索、一键复制。

🖥️

服务器管理

集中管理服务器连接信息，IP、域名、账户、密码一目了然，一键复制全部信息。

📁

智能分组

密码和服务器独立分组系统，灵活组织你的数字资产，查找更便捷。

🎨

现代 UI 设计

简洁直观的界面，支持亮色/暗色主题切换，1200×800 宽敞窗口，高效操作。

🔓

主密码保护

单一主密码解锁所有数据，采用 PBKDF2 密钥派生，暴力破解几乎不可能。

v1.0.5 新增
📄

文本对比

左右分栏对比两段文本，精确到字符级别的差异高亮，支持行级/字符级模式，可上传文本文件。

v1.0.5 新增
🔧

JSON 格式化

一键格式化、压缩、修复 JSON，语法高亮 + 树形视图可折叠展开，支持上传文件和下载结果。

v1.0.6 新增
🔤

Base64 编解码

文本与 Base64 互转，支持 URL Safe 模式，可直接上传图片生成 Base64，解码时实时预览图片。

v1.0.6 新增
🕐

时间戳转换

实时显示当前时间戳，时间戳与日期格式互转，支持秒/毫秒自动识别，展示全球 8 大时区对比。

v1.0.6 新增
#️⃣

哈希计算

支持 MD5、SHA-1、SHA-256、SHA-384、SHA-512，可对文本或文件计算哈希，内置哈希校验功能。

v1.0.6 新增
🔗

URL 字段支持

密码条目新增 URL 字段，卡片直接显示网址并可一键跳转，添加/编辑时一并录入，更完整的账号管理。

无需打开客户端，浏览器内直接管理密码

MV3 新规范

采用 Chrome 最新的 Manifest V3 规范，性能更好，权限管理更安全可控。

✨

自动填充

自动识别登录页面，一键填充账号密码，告别手动输入的烦恼。

🔘

浮动按钮

页面右下角显示浮动按钮，快速访问保存的密码，随时随地一键复制。

💾

本地存储

数据存储在浏览器本地，不上传云端，安全可控，支持导出/导入备份。

🔍

快速搜索

支持按网站名称、账户快速搜索，一键复制账户或密码。

简洁界面

亮色主题设计，简洁清爽，操作直观，安装即用，无需配置。
