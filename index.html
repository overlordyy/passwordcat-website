为什么坚持本地化？

在线工具方便，但代价是数据暴露风险

⚠️

在线工具的隐患

  • 密码、密钥粘贴到在线工具，数据经过对方服务器
  • 浏览器历史、剪贴板可能被恶意扩展读取
  • 在线 Base64/哈希/JSON 工具无法保证不记录输入内容
  • 敏感数据一旦上传，你无法控制其去向
VS
✅

PasswordCat 的方式

  • 所有功能在本地运行，零网络请求
  • Base64、哈希、文本对比、JSON 格式化全部离线计算
  • 密码数据 AES-256-GCM 加密存储，主密码不离开设备
  • 开源代码，任何人都可以审计，没有后门
🔑
私钥、Token 怎么处理？

用 PasswordCat 的备注字段存储，加密保存在本机，而不是粘贴到在线工具。

📄
需要对比配置文件差异？

用内置文本对比功能，文件内容不会离开你的电脑，即使是涉密的生产环境配置。

#️⃣
需要校验文件哈希或计算签名？

内置哈希工具支持文件拖入计算，MD5/SHA256 结果本地生成，不经过任何第三方。

客户端核心功能

安全、高效的桌面密码管理 · 开发者工具箱

端到端加密

采用 AES-256-GCM 加密算法，所有数据本地存储，无需联网，杜绝数据泄露风险。

密码管理

安全存储网站、应用的账号密码，支持分组管理、快速搜索、一键复制。

服务器管理

集中管理服务器连接信息，IP、域名、账户、密码一目了然，一键复制全部信息。

智能分组

密码和服务器独立分组系统，灵活组织你的数字资产，查找更便捷。

现代 UI 设计

简洁直观的界面，支持亮色/暗色主题切换，1200×800 宽敞窗口，高效操作。

主密码保护

单一主密码解锁所有数据，采用 PBKDF2 密钥派生，暴力破解几乎不可能。

哈希计算

SQL 格式化

粘贴带占位符的 SQL 语句，自动识别 ?、$1、:name、#{name} 等参数，填入参数值后一键格式化并替换，结果语法高亮，支持 MySQL / PostgreSQL / SQLite 多方言。

证书解码

支持 PEM、DER、CER、CRT、PFX/P12 多种格式证书解析，自动识别国密 SM2/SM3 证书，展示主体、颁发者、有效期、公钥算法、SAN、指纹等完整信息，一键复制，离线解析。

JWT 工具

三合一 JWT 工具：解码自动拆解 Header/Payload/Signature；编码支持 HS256/384/512 签名；验证离线校验签名有效性，标注过期状态。

v1.1.7 新增

正则表达式解析

逐 token 解析正则含义：锚点、量词、字符类、捕获组、断言、转义序列等；支持测试文本实时匹配高亮，自动猜测正则用途。
